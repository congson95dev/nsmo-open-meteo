# Frontend Tech Stack

- Framework: React (functional components + hooks)
- Routing: react-router-dom
- State management: React Context or lightweight store (your choice)
- HTTP client: fetch or axios
- Chart library: choose ONE suitable for line chart (e.g. Recharts, Chart.js, or ECharts)
- Password hashing: use browser-compatible crypto library (e.g. crypto-js or Web Crypto API)

General rules:
- Clean, readable code
- Reusable components
- No class components

# Project Overview

Build a frontend web application for wind power forecasting.

Backend and database are already implemented:
- Backend: n8n (acts as API server)
- Database: Google Sheets

Frontend responsibility:
- React-based SPA
- Authentication (login + register)
- Configuration input
- Data visualization (line chart)
- Multi-language support (Vietnamese & English)

Important constraints:
- Do NOT implement backend logic
- Do NOT suggest backend architecture
- Only consume APIs
- Save data ONLY when user explicitly clicks Save or Submit

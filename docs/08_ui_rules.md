# UI / UX Rules

- Clean and simple layout
- Desktop-first, responsive is a plus
- Disable buttons while loading
- Show clear error messages
- Do not auto-save any data
- Only save on explicit user action
- Keep chart color mapping consistent:
  - Wind speed column color must match power-vs-wind comparison line color

Navigation:
Login → Chart
Config → Chart
Register → Login

Top bar:
- Keep main menu at top-right position
- On Configuration and Chart pages, show `Nhà máy Điện gió Tân Hợp` in the center area of top bar
